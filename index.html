Brite

The “Brite” Idea is a static website/blog generator. It is both easy to use and yet versitle, supporting the following most excellent features:

Multi-Format Templates

Template can utilize a variety of rendering formats in the same document.

In-Document Metadata

YAML front matter can be used as template variables.

Shadow File Generation

Templates and the static files generated from them can reside side-by-side.

Flexible Stubbing

Or static files can be placed in a location dependent on document properties.

Getting started is easy. Read this basic tutorial and you'll be on off and running with your own "brite" ideas.
